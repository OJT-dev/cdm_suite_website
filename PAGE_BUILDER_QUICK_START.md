## Tips for Success

### 1. Keep It Simple
- Start with 3-5 sections
- Don't overcrowd the page
- Use whitespace effectively

### 2. Use High-Quality Images
- Use image URLs (upload to CDN first)
- Recommended size: 1920x1080 for heroes
- Always add descriptive alt text

### 3. SEO Best Practices
- Fill in meta title & description
- Use descriptive page slugs
- Add relevant keywords

### 4. Test Before Publishing
- Always preview your page
- Check on mobile (use browser tools)
- Test all links and buttons

### 5. Organize Sections
- Use up/down arrows to reorder
- Put most important content first
- End with a CTA

## Troubleshooting

### Page not showing up?
- Check status is "Published" (not "Draft")
- Verify slug doesn't conflict with existing routes
- Clear your browser cache

### Images not loading?
- Verify image URL is accessible
- Use HTTPS URLs only
- Check image file size (< 1MB recommended)

### Sections not saving?
- Check you clicked "Save & Publish"
- Verify you have admin access
- Check browser console for errors
